# 13 Years: A Friendship Appreciation Website

A commemorative, interactive website crafted to celebrate a remarkable 13-year friendship with Heet. Designed to be a visual masterpiece, this project serves as a digital scrapbook and an interactive memory lane, honoring over a decade of shared experiences.

## 🌐 Live Demo
Experience the live website here: [thvariableyt.github.io/13-years/](https://thvariableyt.github.io/13-years/)

## ✨ Features
* **3D Coverflow-Style Photo Album:** An interactive and visually engaging 3D carousel that allows users to smoothly flip through cherished memories and photographs.
* **AI Memory Forge:** A dedicated section designed to present nostalgic moments in a unique, dynamically generated format.
* **Responsive & Interactive UI:** Built with smooth transitions and a carefully crafted user interface to ensure exploring past memories is a seamless experience across devices.

## 🛠️ Tech Stack
* **HTML5:** Core structure and layout of the webpage.
* **CSS3:** Styling, animations, and the foundation for the 3D visual effects.
* **JavaScript:** Logic for the 3D coverflow interactivity and dynamic content rendering.
* **XML:** Used for structuring and managing specific data elements behind the scenes.

## 📂 Repository Structure
* `index.html` - The main entry point and structure of the website.
* `heet1.jpg` to `heet6.jpg` - The primary image assets featured in the 3D photo album.
* `.gitattributes` - Standard Git configuration file.
